# Week 15 — Capstone Presentations & Review

## Format
- Team presentations of the [term project](../../project/README.md): attack → root cause → fix, demoed live.
- Course-wide review and a closing CTF-style challenge covering the whole term.

## Bring
- Threat model, vulnerability report (CWE/OWASP mapped), remediated code, SBOM + signed artifact, and your CI pipeline.
- A 10-minute demo + 5-minute Q&A.

## Closing CTF
A short capture-the-flag mixing web, API, supply-chain, and LLM challenges from Weeks 1–14. Individual, open-notes.
